Código-fonte/Pré-formatação

O comando document.getElementById é usado para selecionar um elemento pelo seu ID. Para tranformar em fonte monoespaçada, você pode usar a tag <code>.

Dá para mesclar com <pre> e <code>

Um texto que ficaria assim:
function exemplo() { console.log("Hello, World!"); }

Pode ficar assim:

    console.log("Hello, World!");
}
